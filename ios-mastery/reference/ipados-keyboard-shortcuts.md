# iPadOS hardware-keyboard shortcuts + modifier legend

For iPad with a hardware keyboard. Hold **⌘** in any app to summon the on-screen shortcut HUD.
Populated by Part 05 (iPadOS). Modifier legend:

| Glyph | Key |
|---|---|
| ⌘ | Command |
| ⌥ | Option |
| ⌃ | Control |
| ⇧ | Shift |
| 🌐 | Globe (Fn) |
| ↩ | Return |
| ⎋ | Escape |
