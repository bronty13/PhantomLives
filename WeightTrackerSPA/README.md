# WeightTrackerSPA

A single-file weight-tracking web app. Open `weighttracker.html` in any
browser — no build, no server, no dependencies beyond a CDN load of
Chart.js. Add weigh-in entries, see a trend chart, basic stats, and tweak
units/goal in Settings. All data is stored locally in the browser via
`localStorage` (nothing leaves the machine).

## Usage

```sh
open weighttracker.html
```
